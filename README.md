# FairSplit 💸

FairSplit helps groups split restaurant bills fairly.

Instead of dividing the bill equally, each person pays only for the items they ordered.

## Features

- Upload receipts
- Assign items to people
- Fair bill splitting
- Modern responsive interface

## Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- Vercel

## Live Demo

https://fairsplit-chi.vercel.app/

## Problem

When dining in groups, people often split the bill equally even though everyone ordered different amounts.

FairSplit calculates exactly what each person owes.
